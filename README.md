# Analisis de Datos — Negocio de Repuestos Automotrices
## Python para Ciencias de Datos | Proyecto Real

---

### Descripcion del proyecto

Analisis exploratorio completo de 2 años de datos reales (Sep 2023 - Sep 2025)
de un negocio de repuestos automotrices en Costa Rica.

El objetivo del analisis es extraer inteligencia de negocio para
fundamentar la apertura de un nuevo local comercial en el sector automotriz.

---

### Tecnologias utilizadas

- Python 3.x
- pandas 2.2.3
- numpy 2.1.3
- matplotlib
- seaborn
- openpyxl
- JupyterLab

---

### Contenido del analisis

| Modulo | Descripcion |
|---|---|
| Carga y exploracion | Lectura de 8 hojas Excel con 21,928 registros |
| Limpieza de datos | Tipos de datos, nulos, columnas derivadas |
| Analisis descriptivo | Estadisticas generales de ventas |
| Analisis temporal | Tendencias por dia, mes y año |
| Analisis de vendedores | Performance y evolucion por año |
| Analisis de clientes | Frecuencia, ticket promedio, clasificacion VIP |
| Analisis de inventario | Rotacion, stock critico, valor en bodega |
| Analisis de proveedores | Ranking por rendimiento combinado |
| Punto de equilibrio | Calculo de ventas minimas y escalera salarial |
| Articulos criticos | Identificacion de repuestos de emergencia |

---

### Hallazgos principales

- Ventas totales en 2 años: 192,218,919 colones costarricenses
- Ticket promedio por transaccion: 11,080 colones
- Mejor mes historico: Febrero 2025 con 9,889,646 colones
- Total clientes unicos atendidos: 3,077
- Catalogo de 10,120 articulos en 37 categorias
